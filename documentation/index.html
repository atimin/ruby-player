Installation

The RubyPlayer is library written in pure Ruby. For installation on client machine you should have only installed ruby >= 1.9.2 and rubygems:

$ gem install ruby-gems

The RubyPlayer is developing and testing for player-3.1.0 version which is in developing process too. You can get source code from here. You may get info about installation player server here.
